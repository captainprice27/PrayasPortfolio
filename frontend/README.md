# Portfolio Frontend

React portfolio built with Vite.

## Quick Start

```bash
npm install
npm run dev
```

## Component Structure

- `Navbar` - Navigation with theme toggle
- `Hero` - Landing section
- `About` - Personal info
- `Skills` - Technical skills
- `JourneyTimeline` - Education/work history
- `Projects` - Project showcase
- `Contact` - Contact form
- `Footer` - Site footer

## Customization

Edit the configuration objects at the top of each component file.
