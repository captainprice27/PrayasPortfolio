# Portfolio Backend

Express API for the portfolio application.

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and update the values.

## API Routes

- `GET /api/portfolio` - All data
- `GET /api/skills` - Skills only
- `GET /api/projects` - Projects only
- `GET /api/journey` - Journey timeline
